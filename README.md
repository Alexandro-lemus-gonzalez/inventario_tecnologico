# 🖥️ Inventario Tecnológico

Aplicación web desarrollada con Django para la gestión y control de activos tecnológicos dentro de una organización.

---

## 🚀 Funcionalidades principales

✅ Registro de activos tecnológicos  
✅ Edición y eliminación (CRUD completo)  
✅ Dashboard con estadísticas en tiempo real  
✅ Clasificación por categoría, estado y ubicación  
✅ Historial de movimientos de activos  
✅ API REST documentada con Swagger  
✅ Interfaz moderna con Bootstrap 5  

---

## 📊 Dashboard

El sistema incluye un panel de control donde se visualiza:

- Total de activos
- Valor total del inventario
- Activos en mantenimiento
- Activos por categoría

---

## 🛠️ Tecnologías utilizadas

- 🐍 Python  
- 🌐 Django  
- 🗄️ MySQL  
- 🎨 Bootstrap 5  
- 🔗 Django REST Framework  
- 📄 Swagger (Documentación API)

## ⚙️ Instalación y ejecución

1. Clonar el repositorio:

```bash
git clone https://github.com/Alexandro-lemus-gonzalez/inventario_tecnologico.git
```

2. Entrar al proyecto:

```bash
cd inventario_tecnologico
```

3. Crear entorno virtual:

```bash
python -m venv venv
```

4. Activar entorno:

```bash
venv\Scripts\activate
```

5. Instalar dependencias:

```bash
pip install -r requirements.txt
```

6. Ejecutar migraciones:

```bash
python manage.py migrate
```

7. Iniciar servidor:

```bash
python manage.py runserver
```
